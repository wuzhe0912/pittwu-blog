前工作的前端團隊都使用 SourceTree 進行版控，相對較少使用指令操作 git，為了方便日後終端機操作，記錄一下指令功能。
<!--more-->
- 初始化git專案
```
git init
```
- 將檔案加入版本控制
```
git add .
```
- 建立 develop 分支
```
git branch develop
```
- 切換分支到 develop
```
git checkout develop
```
- 推送分支到遠端
```
git push -u origin develop
```
